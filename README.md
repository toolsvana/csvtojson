# CSV to JSON Converter

A browser-based tool that converts CSV data to JSON format. Built with Next.js and runs entirely in your browser.

ðŸ”— **Live Demo:** [CSV to JSON](https://toolsvana.com/tool/json-csv-xml)

## What it does

This tool parses CSV (Comma-Separated Values) data and converts it to JSON format. Handles quoted values, commas, and special characters correctly.

## Key features

- Convert CSV to JSON instantly
- Smart CSV parsing
- Prettify JSON output option
- Sample data loader
- Copy results with one click
- Works offline
- No data sent to servers

## How to use

1. Paste your CSV data or load sample
2. Choose prettify option if desired
3. Click "Convert to JSON"
4. Copy the JSON output

## Technical details

Built with Next.js 16, TypeScript, and Tailwind CSS. Uses custom CSV parser for accurate conversion.

## Running locally

```bash
npm install
npm run dev
```
